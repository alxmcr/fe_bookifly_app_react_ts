# Bookifly

Travel safely, comfortably, and quickly with your family.

## Run Locally

Clone the project

```bash
  git clone https://github.com/alxmcr/fe_bookifly_app_react_ts
```

Go to the project directory

```bash
  cd fe_bookifly_app_react_ts
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
